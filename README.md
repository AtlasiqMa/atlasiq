#  AtlasIQ

> End-to-end data platform tracking Morocco's tourism pulse — from raw ingestion to real-time insights.

## Overview

AtlasIQ is a full data engineering platform that ingests, processes, and models
Morocco tourism data — flights, weather, hotel demand, and visitor trends —
served through a live analytics web app.

## Repositories

| Repo | Description |
|---|---|
| [atlasiq-pipelines](https://github.com/AtlasiqMa/atlasiq-pipelines) | Airflow DAGs, Kafka, Spark jobs |
| [atlasiq-warehouse](https://github.com/AtlasiqMa/atlasiq-warehouse) | dbt models, tests, documentation |
| [atlasiq-webapp](https://github.com/AtlasiqMa/atlasiq-webapp) | React frontend + API |
